# 🔹 Unit 2: Understanding Cloud Architecture and Implementing Different Frameworks

## 📌 Overview
This unit focuses on cloud architecture and the frameworks used to design, deploy, and manage cloud systems. It introduces key methodologies that support structured and scalable cloud solutions.

---

## 📚 Topics Covered
- Cloud architecture principles  
- TOGAF framework  
- Software Development Life Cycle (SDLC)  
- Infrastructure as Code (IaC)  

---

## 🎯 What I Learned
In this unit, I learned how cloud systems are structured using architectural frameworks such as TOGAF and SDLC. I also gained an understanding of Infrastructure as Code (IaC), which enables automation and efficient management of cloud infrastructure.

---

## 🔍 Reflection
This unit highlighted the importance of structured design in cloud environments. While IaC improves efficiency and reduces manual errors, it also introduces risks if not properly managed. Errors in scripts can affect entire systems, which emphasises the need for testing and governance.

---

## 🛠️ Skills Developed
- Understanding cloud architecture frameworks  
- Applying Infrastructure as Code concepts  
- Analysing system design and scalability  

---

## 🚀 Practical Insight
The use of IaC and structured frameworks is essential in industry, as it allows organisations to automate deployments and maintain consistency across cloud environments.
