# 🔹 Unit 1: Introduction to Cloud Fundamentals and Services

## 📌 Overview
This unit introduces the fundamental concepts of cloud computing, including its definition, characteristics, and evolution. It provides a strong foundation for understanding how cloud technologies are transforming modern IT infrastructure.

---

## 📚 Topics Covered
- Definition and characteristics of cloud computing  
- Cloud service models (IaaS, PaaS, SaaS)  
- Benefits of cloud computing (scalability, flexibility, cost-efficiency)  
- Challenges (security, vendor lock-in, compliance)  

---

## 🎯 What I Learned
In this unit, I learned the core principles of cloud computing and how it differs from traditional IT infrastructure. I gained a clear understanding of service models such as IaaS, PaaS, and SaaS, and how organisations use them to manage resources more efficiently.

---

## 🔍 Reflection
This unit helped me realise that cloud computing is not just a technical innovation but a strategic business tool. While it offers significant advantages such as scalability and cost reduction, it also introduces risks such as data security concerns and dependency on cloud providers. This highlighted the importance of evaluating cloud solutions carefully before implementation.

## 🛠️ Skills Developed
- Understanding cloud service models  
- Identifying benefits and risks of cloud adoption  
- Analysing real-world cloud use cases

## 🚀 Practical Insight
The knowledge gained in this unit is essential for selecting appropriate cloud solutions in real-world scenarios. It provides a foundation for understanding more advanced topics such as cloud architecture and automation.
